# Linux.do 归档助手扩展

这是项目内置的浏览器扩展目录，建议通过“加载已解压的扩展程序”方式使用。

## 主要能力

- 导出当前打开的 Linux.do 帖子
- 自动补齐分页楼层
- 可选生成 PDF
- 支持楼层范围导出
- 显示导出进度
- 导出完成后打开输出目录

## 使用前提

先在项目根目录启动本地桥：

```powershell
uv run python .\local_bridge_server.py
```

## 加载方式

1. 打开 `chrome://extensions/`
2. 打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录 `browser-extension`

## 使用流程

1. 打开一个 `https://linux.do/t/...` 帖子页
2. 点击扩展图标
3. 选择是否生成 PDF
4. 如有需要，在“高级选项”里填写楼层范围
5. 点击“导出当前帖子”
6. 等待进度条走完
7. 导出完成后点击“打开最近一次输出目录”

## 当前限制

- 只处理当前标签页
- 只支持 `linux.do`
- 依赖本地桥服务
- 进度条主要反映本地桥处理阶段，不是网络抓取真实百分比
