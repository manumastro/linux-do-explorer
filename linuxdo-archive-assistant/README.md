## 🧭 What this app does

linuxdo-archive-assistant helps you save a Linux.do topic to your computer as:

- Markdown text for easy reading and editing
- A PDF that keeps images in place
- A local copy you can reuse with AI tools later

It is built for people who want to keep a topic after it changes online or disappears later.

## 💻 What you need

Use a Windows PC with:

- Windows 10 or Windows 11
- A working internet connection
- Enough free space for the topic, images, and PDF
- Permission to download and run files on your computer

For best results, keep Chrome, Edge, or another modern browser installed.

When you open the app for the first time:

2. Choose where you want the files saved.
3. Select Markdown export if you want a text copy.

## 🛠️ Simple workflow

A clean process helps keep your archive organized:

1. Open the topic in Linux.do.
2. Copy the topic link from the address bar.
3. Paste the link into linuxdo-archive-assistant.
4. Choose the output folder.
5. Export the topic.
6. Check the Markdown file and PDF.
7. Move the folder to your long-term archive.

## 🔍 Tips for better results

- Use the full topic URL
- Keep the original folder names simple
- Do not close the app until the export ends
- Use a folder with enough space for images
- Keep the output folder separate from Downloads
- Rename the saved folder with the topic title and date

Example folder name:

## 🖼️ Why image-preserving PDF matters

Some exports lose screenshots, diagrams, or inline images. This app keeps those images in the PDF so the archive stays useful later.

That helps when you need:

- The full topic layout
- Visual context from the discussion
- Proof of what was shown at the time
- A file you can open on any Windows PC
