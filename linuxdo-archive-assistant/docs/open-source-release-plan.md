## 当前定位

当前版本定位为：

- 可公开展示的原型版本
- 可供 Linux.do 用户手动安装和使用
- 重点强调“稳定、可解释、低风险”

## 当前使用闭环

1. 启动本地桥
2. 浏览器加载 `browser-extension`
3. 打开 Linux.do 帖子页
4. 点击“导出当前帖子”
5. 在扩展里查看阶段进度
6. 导出完成后打开结果目录

## 发布建议

当前建议以 `v0.2.0` 作为开源首个可用原型版本：

- 核心功能已可用
- 用户能看到进度
- 高级功能不会干扰默认使用流
- 手动启动本地桥的模型更稳定、更易解释

## 发布形态建议

建议采用双轨制发布：

- 源码仓库：面向开发者、跨平台用户
- Windows 便携版：面向普通 Windows 用户

Windows 便携版建议内容：

- `linuxdo-archive-bridge.exe`
- `browser-extension/`
- 启动/停止/打开目录脚本
- 便携版使用说明

这样既保留开源透明度，也降低普通用户的使用门槛。
